## 개발 환경 및 오픈 SW
* IDE: Visual Studio Code
* Frontend: HTML, CSS, JavaScript <br>
* Backend: Flask, Python <br>
* Database: Firebase <br>

## 디렉토리 구조
flask_projectteam4/ <br>
├── authentication/       `Firebase Authentication 및 Realtime Database 연동` <br>
├── static/              `정적 파일 (CSS, JavaScript, 이미지)` <br>
├── templates/           `HTML 템플릿 파일` <br>
├── app.py               `백엔드 핵심 로직` <br>
├── database.py          `데이터베이스 연결 및 모델 정의` <br>
└── README.md            `프로젝트 설명 문서` <br>

## 특징
* 프론트엔드와 백엔드 연동 => 사용자가 입력한 데이터가 서버와 DB에 실시간으로 반영됨
* `다크모드 기능`: 사용자의 설정에 따라 사이트의 전체 테마가 자동으로 전환됨
