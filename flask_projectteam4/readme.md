이 프로젝트는 HTML 기반의 프론트엔드와 Flask 백엔드를 연동하여, 사용자가 입력한 데이터가 서버와 데이터베이스에 실시간으로 반영되는 과정을 자연스럽게 보여주는 웹 서비스입니다. 특히 시연 영상에서는 상품을 등록하면 해당 정보가 즉시 데이터베이스에 저장되고, 웹 페이지에서도 즉시 확인할 수 있는 흐름을 중점적으로 보여주어, 프론트엔드와 백엔드 간의 연동이 원활하게 이루어진다는 점을 강조하고자 했습니다.

또한 다크모드 기능을 포함하고 있어, 사용자의 설정에 따라 사이트의 전체 테마가 자동으로 전환되도록 구현되었습니다. 이러한 UI/UX 요소로 프로젝트의 완성도를 높이려고 했습니다.

프로젝트 전체 디렉터리는 직관적으로 구성되어 있으며, templates 폴더에는 HTML 템플릿이, static 폴더에는 CSS·JS·이미지 파일이 정리되어 있습니다. Flask 백엔드의 핵심 로직은 app.py에 구현되어 있으며, 데이터베이스 연동을 담당하는 모듈은 별도로 분리하여 유지보수성과 가독성을 높였습니다. 각 코드 파일에는 주요 기능과 흐름을 이해하기 쉽도록 주석을 작성해 두었으며, 오픈 소스 라이브러리 또한 적절히 활용하여 UI 구성과 서버 기능 구현에 도움이 되도록 했습니다.

이 프로젝트는 실제 데이터 흐름이 동작하는 웹 서비스의 구조를 보여주는 것에 목적을 두었으며, 프론트엔드–백엔드–데이터베이스가 유기적으로 연결되는 구조를 명확하게 설명할 수 있도록 구성했습니다. 이를 통해 사용자 경험을 강화하고, 오픈 소스를 적극적으로 활용하여 효율적이고 확장 가능한 웹 애플리케이션을 구현하고자 했습니다.
